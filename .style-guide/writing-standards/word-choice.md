# Word Choice

| Avoid                 | Use instead                                                            |
| --------------------- | ---------------------------------------------------------------------- |
| webshop               | storefront, frontend (in the context of Emporix B2B Commerce Frontend) |
| app user              | customer                                                               |
| should                | must (for requirements), can (for optional behavior)                   |
| click                 | select (generic UI action)                                             |
| whitelist / blacklist | allowlist / blocklist                                                  |
| e-mail                | email                                                                  |
| logon (noun)          | login or sign-in (noun), log in or sign in (verb)                      |
| setup (verb)          | set up                                                                 |
| auth                  | authentication (unless in code identifiers)                            |
| payload data          | payload                                                                |
| API's                 | APIs                                                                   |
