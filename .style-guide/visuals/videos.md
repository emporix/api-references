# Videos

To create and edit video content, use the Filmora tool. 

* To add a video hosted on external platform to documentation, use the **embed URL** block element.

Example:

`{% embed url="URL_HERE" %}`

* To add a playable video and host it directly at GitBook, follow this instruction: [Adding a playable video](https://gitbook.com/docs/guides/editing-and-publishing-documentation/upload-and-embed-a-playable-video-into-your-gitbook-docs).

Example:

```
{% embed url="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FYAak0Z7i2XjNSviBuJeI%2Fuploads%2FOLFWQe2C7wbl6Pa9pSCL%2Fnavigation_modules-1.mp4?alt=media&token=32de470c-27e3-45e6-b048-2d0053b4f955" %}
Modules Manager options
{% endembed %}
```
