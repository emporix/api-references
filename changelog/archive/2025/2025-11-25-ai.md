# 2025-11-25: AI Service - Agent Logs

## Overview

Added a new API for retrieving agent logs, allowing you to monitor and debug agent requests and sessions.

## Added endpoints

| Endpoint                                                                                                                                                                                                    | Description                                 |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------|
| [Listing agent requests](https://developer.emporix.io/api-references/api-guides/artificial-intelligence/ai-service/api-reference/agent-logs#get-ai-service-tenant-agentic-logs-requests)                    | It allows retrieving a list of agent request logs with pagination, sorting, and filtering support. |
| [Retrieving agent request by ID](https://developer.emporix.io/api-references/api-guides/artificial-intelligence/ai-service/api-reference/agent-logs#get-ai-service-tenant-agentic-logs-requests-requestid) | It allows retrieving a single agent request log by its ID.                                                |
| [Searching agent requests](https://developer.emporix.io/api-references/api-guides/artificial-intelligence/ai-service/api-reference/agent-logs#post-ai-service-tenant-agentic-logs-requests-search)          | It allows searching for agent request logs based on the provided query.                                  |
| [Listing agent sessions](https://developer.emporix.io/api-references/api-guides/artificial-intelligence/ai-service/api-reference/agent-logs#get-ai-service-tenant-agentic-logs-sessions)                    | It allows retrieving a list of agent session logs with pagination, sorting, and filtering support.      |
| [Retrieving agent session by ID](https://developer.emporix.io/api-references/api-guides/artificial-intelligence/ai-service/api-reference/agent-logs#get-ai-service-tenant-agentic-logs-sessions-sessionid) | It allows retrieving a single agent session log by its ID, including all associated log messages.      |
| [Searching agent sessions](https://developer.emporix.io/api-references/api-guides/artificial-intelligence/ai-service/api-reference/agent-logs#post-ai-service-tenant-agentic-logs-sessions-search)            | It allows searching for agent session logs based on the provided query.                                 |

## Known problems

There are no known problems.
