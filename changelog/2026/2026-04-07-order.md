# 2026-04-07: Order Service - historical transitions endpoint for customer-managed orders

## Overview

Added documentation for a new customer-managed endpoint to retrieve historical status transitions of a specific order.

The update includes:

- New endpoint `GET /order-v2/{tenant}/salesorders/{orderId}/historical-transitions`
- Response schema for transition history with metadata
- Example payload for the new response
