# RELEASE_DATE: AI Service - RAG_EMPORIX support for custom entity types

## Overview

The tool of `RAG_EMPORIX` type allow specifying the custom entity type by using `entityType` field.

## Updated endpoints

| Endpoint                                                                                                                                                     | Description                                                                 |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------|
| [Listing tools](https://developer.emporix.io/api-references/artificial-intelligence/ai-service/api-reference/agent-analytics#GET-ai-list-tools)              | Field `entityType` of `RAG_EMPORIX` tool type supports custom entity types. |
| [Searching tools](https://developer.emporix.io/api-references/artificial-intelligence/ai-service/api-reference/agent-analytics#POST-ai-search-tools)         | Field `entityType` of `RAG_EMPORIX` tool type supports custom entity types. |
| [Retrieving tool by ID](https://developer.emporix.io/api-references/artificial-intelligence/ai-service/api-reference/agent-analytics#GET-ai-retrieve-tool)   | Field `entityType` of `RAG_EMPORIX` tool type supports custom entity types. |
| [Upserting tool](https://developer.emporix.io/api-references/artificial-intelligence/ai-service/api-reference/agent-analytics#PUT-ai-upsert-tool)            | Field `entityType` of `RAG_EMPORIX` tool type supports custom entity types. |
| [Partially updating tool](https://developer.emporix.io/api-references/artificial-intelligence/ai-service/api-reference/agent-analytics#PATCH-ai-update-tool) | Field `entityType` of `RAG_EMPORIX` tool type supports custom entity types. |

There are no known problems.
