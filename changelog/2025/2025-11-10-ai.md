# 2025-11-10: AI Service - Export/Import agents

## Overview

We added a new API for exporting and importing AI agents.
We also extended the jobs API with the new fields `type`, 'exportResult' and `importResult`.

## Added endpoints

| Endpoint                                                                                                                                                     | Description                    |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------|
| [Exporting agents](https://developer.emporix.io/api-references/api-guides/artificial-intelligence/ai-service/api-reference/agent-chat#post-ai-export-agents) | It allows exporting AI agents. |
| [Importing agents](https://developer.emporix.io/api-references/api-guides/artificial-intelligence/ai-service/api-reference/agent-chat#post-ai-import-agents) | It allows importing AI agents. |

## Modified endpoints

| Endpoint                                                                                                                                                      | Description                                                                                |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [Listing available jobs](https://developer.emporix.io/api-references/api-guides/artificial-intelligence/ai-service/api-reference/agent-chat#get-ai-list-jobs) | New fields have been added to the Job response: `type`, 'exportResult' and `importResult`. |
| [Searching jobs](https://developer.emporix.io/api-references/api-guides/artificial-intelligence/ai-service/api-reference/agent-chat#post-ai-search-jobs)      | New fields have been added to the Job response: `type`, 'exportResult' and `importResult`. |
| [Retrieving available job](https://developer.emporix.io/api-references/api-guides/artificial-intelligence/ai-service/api-reference/agent-chat#get-ai-job)     | New fields have been added to the Job response: `type`, 'exportResult' and `importResult`. |

## Known problems

There are no known problems.
