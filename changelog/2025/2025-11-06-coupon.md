# 2025-11-06: Coupon Service - removal of deprecated endpoints

## Overview

All previously deprecated endpoints are now
removed ([2025-03-28: Coupon Service - deprecations](../2025/2025-03-28-coupon-deprecations.md)).

| Endpoint                                                                                                                                                                                                                                    | Description                                                                                                                                                                 |
