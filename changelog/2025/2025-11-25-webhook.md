# 2025-11-25: Webhook Service - update category event schema

## Overview

Category event schemas were updated to include support for classification categories and their associated mixins.

## Added new fields

| Event                  | Description                                                                           |
|------------------------|---------------------------------------------------------------------------------------|
| `category.created`     | Support for `ownClassificationMixins` and `classificationMixins` added.     |
| `category.updated`     | Support for `ownClassificationMixins` and `classificationMixins` added.     |

## Known problems

There are no known problems.
