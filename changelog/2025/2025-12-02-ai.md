# 2025-12-02: AI Service - Streamable HTTP MCP servers

## Overview

The AI Service has been extended to support a new MCP server transport protocol: `streamable_http`. This option enables streamable
HTTP-based communication between agents and custom MCP server implementations alongside the existing `sse` transport.
